Green Basket

Build RESTful Api for An Online Vegetable Sales Application

The application allows customers to easily purchase fresh, locally sourced vegetables from their phone or computer. With a user-friendly interface, customers can browse and select from a wide variety of fruits and vegetables, customize their order, and choose a delivery or pickup option that works for them. This repository contains the source code for the backend of the application, which is built using Java and the Spring Framework. It handles all the business logic and communicates with a MySQL database. The backend also exposes RESTful endpoints for the frontend to consume.
